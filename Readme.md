# EduBot - AI Personalized Study Assistant

## SDG Chosen
SDG 4 - Quality Education

## Project Objective

EduBot is an AI-powered educational chatbot that provides instant academic assistance, study plans, topic summaries and educational resources to students.

## Features

- Doubt Solver
- Study Plan Generator
- Topic Summarizer
- Resource Recommendation
- Revision Helper

## Technologies Used

- Botpress
- Artificial Intelligence
- GitHub

## How to Use

1. Open the chatbot.
2. Ask academic questions.
3. Generate study plans.
4. Summarize topics.
5. Get learning resources.

## Future Scope

- Voice interaction
- Multi-language support
- Mobile application
- Learning analytics dashboard
